# Sección 4. Documento de Diseño de Software

RMRS usa una arquitectura MVC con PHP, Apache/XAMPP y MySQL/MariaDB. El router resuelve la solicitud, el middleware comprueba sesión y rol, los controladores coordinan la operación, los modelos acceden a la base mediante PDO y las vistas renderizan la interfaz.

El [Documento de Diseño de Software completo](sdd.md) contiene arquitectura, módulos, componentes, modelo de datos, navegación, seguridad, persistencia y evolución. Los diagramas técnicos se encuentran en el [Anexo C](anexos/anexo-c-diagramas.md).
