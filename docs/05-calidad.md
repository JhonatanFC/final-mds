# Sección 5. Plan de calidad del software

El perfil ISO/IEC 25010 prioriza adecuación funcional, fiabilidad y seguridad. La calidad se controla desde el refinamiento con criterios de aceptación, revisión de riesgos, diseño de datos, validación de código y pruebas funcionales.

La matriz de pruebas, actividades SQA, enfoque Shift Left y costo de calidad se documentan en [calidad ISO 25010 y SQA](entrega/05-calidad-iso-sqa.md).
