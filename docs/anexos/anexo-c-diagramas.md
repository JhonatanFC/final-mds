# Anexo C. Diagramas del sistema

El informe final incluye diagramas de casos de uso, arquitectura MVC, actividad de reserva y atención, modelo de dominio y flujo de gestión. Los activos gráficos deben almacenarse en [`docs/images/`](../images/README.md) con la convención indicada allí.

Los diagramas deben reflejar exclusivamente los módulos implementados: reservas, vouchers, recepción, mesas, lista de espera, pedidos, caja, usuarios, dashboard y reportes.
