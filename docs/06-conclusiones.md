# Sección 6. Conclusiones y reflexión crítica

El sistema integra el flujo operativo de D'Barrio Broaster desde la reserva hasta el reporte gerencial. Scrum adaptado permitió organizar las entregas por módulos y acompañarlas con evidencia de calidad y gestión.

Las limitaciones actuales son la ausencia de pasarela de pago real, notificaciones automáticas, aplicación móvil nativa, pruebas automatizadas e integración continua. Las reflexiones individuales y la declaración de autoría se encuentran en [cierre y anexos](entrega/06-cierre-y-anexos.md).
